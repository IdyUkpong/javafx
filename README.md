# Project Overview

This is a simple JavaFX desktop application that collects basic population data through a form.
The application allows users to enter their name, age, and country, and saves the data into a database.


# Features

JavaFX-based graphical user interface (GUI)

Form inputs: Name Age Country

# Technologies Used

Java (JDK 17 / 21 recommended)
JavaFX
Maven
Database (e.g., SQLite / MySQL / PostgreSQL)

# How to Run the Project
1️⃣ Prerequisites

Java 17 or 21

Maven 3.8+

JavaFX dependencies configured in pom.xml

2️⃣ Run the application

From the project root directory:
mvn javafx:run
